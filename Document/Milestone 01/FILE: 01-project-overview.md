Project Name: CompareAI

Description:

CompareAI is a full-stack web platform that allows users to compare responses from multiple AI models simultaneously.

Purpose:

• Compare model output quality
• Compare model speed
• Evaluate model reasoning
• Provide unified interface for multiple providers

Core Concept:

User enters prompt → backend sends prompt to multiple models → frontend displays responses in parallel containers.

Tech Stack:

Frontend:
• React
• TypeScript
• Vite

Backend:
• NestJS
• TypeScript
• Axios
• OpenAI SDK

Architecture type:
Distributed multi-provider orchestration platform
